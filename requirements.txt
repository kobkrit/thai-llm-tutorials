# NOTE ON torch
# -------------
# torch is deliberately NOT pinned here. Colab ships a torch build matched to
# its own CUDA driver, and `pip install torch==x.y.z` over the top is the single
# most common way these notebooks break. Use whatever torch Colab provides
# (>= 2.1 is required for attn_implementation="sdpa").
#
# If you are running locally rather than on Colab, install torch first, from
# https://pytorch.org, matching your CUDA version -- then install this file.
#
# T4 CONSTRAINTS baked into these choices
# ---------------------------------------
#   * NO bfloat16          -> use torch.float16 and fp16=True
#   * NO FlashAttention-2  -> use attn_implementation="sdpa"
# flash-attn is therefore absent from this file on purpose. Adding it will
# either fail to build or fail at runtime on a T4.
# =============================================================================
